Jump & Duck

Instructions

Note: This game plays in the browser tab title! Watch the tab as you play.

Example of how the game looks in your browser tab (Chrome)

  • Arrow Up → Jump
  • Arrow Down → Duck
  • Space → Start / Pause

Good luck! Dodge the spikes:

Player: ■   Bottom spike (jump): ▲   Top spike (duck): ▼

Important Tips

  • Open in a new window with no other tabs so the title is fully visible, as multiple tabs can make it too small to see properly.
  • If you're using Safari, make sure View → Show Tab Bar is enabled.
