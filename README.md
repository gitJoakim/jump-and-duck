# Jump & Duck

[![Play it here](https://img.shields.io/badge/Play-Here-blue?style=for-the-badge)](https://gitJoakim.github.io/jump-and-duck/)

A small browser tab title game where you **jump** (Arrow Up) or **duck** (Arrow Down) to dodge spikes.  
Press **Space** to start/pause.  

> **Note:** This game runs entirely in the browser tab title! Keep an eye on the tab as you play.

## Instructions

- Arrow Up → Jump  
- Arrow Down → Duck  
- Space → Start / Pause  

**Good luck! Dodge the spikes:**  
Player: ■ &nbsp; Bottom spike (jump): ▲ &nbsp; Top spike (duck): ▼

---

## Important Tips

- Open in a **new window with no other tabs** so the title is fully visible, as multiple tabs can make it too small to see properly.  
- If you're using Safari, make sure **View → Show Tab Bar** is enabled.  

*The game is buggy and horrible, have fun :)*
